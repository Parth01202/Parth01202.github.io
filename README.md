# Parth Patel - Portfolio Website

A modern, responsive portfolio website showcasing my experience as a Back-End Java Developer, featuring projects, skills, and professional experience.

## 🚀 Features

- **Modern Design**: Clean, professional design with smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling and active section highlighting
- **Project Showcase**: Detailed descriptions of three major projects (P0, P1, P2)
- **Skills Display**: Organized skill categories with sub-skills
- **Experience Timeline**: Visual timeline of professional experience
- **Contact Information**: Easy-to-access contact details

## 📋 Sections

1. **Hero**: Introduction with call-to-action buttons
2. **About**: Professional summary and highlights
3. **Skills**: Technical skills organized by category
   - Spring
   - React
   - Front-End
   - DevOps
   - Data
4. **Projects**: Three featured projects with detailed outcomes
   - Banking Management System (P0)
   - Expense Reimbursement System (P1)
   - Social Media Microservices Application (P2)
5. **Experience**: Professional work experience at HealthOne Medical
6. **Education**: Academic background and certifications
7. **Contact**: Contact information and social links

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **JavaScript**: Interactive features and smooth scrolling
- **Font Awesome**: Icons
- **Google Fonts**: Inter font family
